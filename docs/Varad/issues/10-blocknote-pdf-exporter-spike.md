## What to build

Evaluate `@blocknote/xl-pdf-exporter` against the paper editor’s question block tree and exam-ready PDF requirements without making it the official export path.

## Acceptance criteria

- [ ] Confirm package licensing implications for project use.
- [ ] Prototype export from the editor BlockNote document to PDF using the XL exporter.
- [ ] Document gaps versus canonical backend-rendered PDF: print fidelity, custom block mappings, editor-only UI removal, page layout, and source-of-truth concerns.
- [ ] Recommend whether BlockNote PDF export is useful for draft preview, official export, or not worth adopting.
- [ ] Official V1 PDF path remains backend-rendered unless a later decision changes it.
