# Issue Tracker

Issues and PRDs for this repo live in GitHub Issues:

- Repo: `animesh0911/qs-paper-generator`
- CLI: use `gh` from inside this clone
- Varad/frontend/editor work: apply label `V`

## Commands

- Create: `gh issue create --title "..." --label V --body "..."`
- Read: `gh issue view <number> --comments`
- List V work: `gh issue list --label V --state open`
- Edit: `gh issue edit <number> --body-file <file>`
- Comment: `gh issue comment <number> --body "..."`

When a skill says to publish a PRD or implementation slice, create a GitHub
issue unless the user asks for local markdown only.
