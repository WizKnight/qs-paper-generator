# Domain Docs

This is a single-context repo.

Before coding or reviewing non-trivial work, read:

- `CONTEXT.md` for domain vocabulary.
- Relevant files in `docs/Varad/` for current editor and AI editor decisions.
- Relevant files in `docs/adr/` if that directory exists.

Use the vocabulary from `CONTEXT.md` in module names, issue titles, test names,
and implementation notes. If a term is missing or ambiguous, resolve it through
`grill-with-docs` before spreading a new synonym through code.
